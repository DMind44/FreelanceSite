Static homepage for a freelance game developer business.

Files:
- index.html
- styles.css
- app.js
- services.txt

How to run locally:
1. Open a terminal in this folder.
2. Run: python -m http.server 3000
3. Visit: http://localhost:3000

Important:
- Do not open index.html directly from the file system if you want services.txt to load.
- Edit services.txt to add, remove, or change services.
