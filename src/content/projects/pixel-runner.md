## About

Pixel Runner is a fast-paced endless runner with dynamically generated levels that respond to the player's skill. The game features retro-inspired visuals with modern gameplay mechanics.

## What I Did

- Programmed the core gameplay loop and physics system
- Implemented the dynamic level generation algorithm
- Built the scoring and leaderboard system
- Optimized performance for mobile devices

## Technical Highlights

The level generation system uses a weighted probability system that adjusts difficulty based on player performance. Obstacles, power-ups, and environmental elements are selected from a pool that shifts based on the current run's metrics.
